¿Qué lenguajes vamos a aprender en Adalab?

HTML

Es el encargado de desarrollar una descripción sobre los contenidos que aparecen como textos
y sobre su estructura, complementando dicho texto con diversos objetos (como fotografías, animaciones, etc). Ver Wikipedia

CSS

Este lenguaje define la apariencia de un documento escrito en un lenguaje de marcado para
poder darle la apariencia que se desee utilizando CSS: colores, espacios entre elementos, tipos de letra, ... separando
de esta forma la estructura de la presentación.Ver Wikipedia

JAVASCRIPT

Es un lenguaje en el que se ejecutan una serie de secuencias de comandos que te permite
implementar funciones complejas en páginas web para que haya interacción entre cliente y usuario.Ver Wikipedia
